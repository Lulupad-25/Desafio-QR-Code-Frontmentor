## O que aprendi

Com esse desafio consegui praticar conceitos importantes de HTML e CSS, principalmente a estruturação de elementos na página e estilização de componentes.

Durante o desenvolvimento aprendi melhor sobre:

- Estrutura básica do HTML
- Organização de elementos com `div`
- Uso de imagens no HTML
- Estilização com CSS
- Centralização de elementos usando Flexbox
- Espaçamentos com `margin` e `padding`
- Bordas arredondadas com `border-radius`
- Aplicação de sombras com `box-shadow`
- Uso de cores e tipografia
- Responsividade básica

Também consegui entender melhor como posicionar elementos na página usando `flex-direction`, além da importância de manter o código organizado e semântico.

Esse projeto me ajudou bastante a ganhar mais confiança com HTML e CSS na prática.
